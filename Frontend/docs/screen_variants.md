## 구현 범위 — 7화면 (관리자 콘솔 제외)

| # | 화면 | 변형 | 라우트 | 핵심 |
|---|------|------|--------|------|
| 01 | 랜딩 / 가입 안내 | **B** 미니멀 히어로 | `/` (비로그인) | 큰 카피 + 풀폭 대시보드 목업 + "왜 에너지캐시백인가" 3-feature |
| 02 | 로그인 / 회원가입 | **A** 좌우 분할 | `/auth/*` | 좌 다크 패널(브랜드) + 우 폼(이메일/비밀번호 + SSO + 회원가입 링크) |
| 03 | 대시보드 (홈) | **C** 분석형 | `/` (로그인 후) | 좌 2/3 차트(주간 + 월별) + 우 1/3 KPI 3개 + 가전별 점유율 바 |
| 04 | 사용량 분석 | **A** 종합 분석 | `/usage` | 기간 세그먼트 + 주간 페어 막대 + 24h 라인 + 가전별 분해 표 + 월별 막대 |
| 05 | 캐시백 | **C** 목표 트래커 | `/cashback` | 월 목표 진행바(현재/예상/목표 마커) + 주간/월별 차트 + 오늘의 미션 |
| 06 | 설정 / 계정 | **A+B+C+D+E 결합** | `/settings/*` | 좌측 200px 사이드바 6 탭 (아래 §06 매핑 참조) |
| 07 | AI 진단 | **A** 진단 요약 | `/insights` | 3 KPI(진단 건수/예상 절약/신뢰도) + 이상 사용 카드 + 추천 표 + 주간 추이 |

**제외**: 08 관리자 콘솔 — Phase 0 범위 외. 후속 페이즈에서 별도 검토.

## 06. 설정 / 계정 — 사이드바 매핑

좌측 200px 사이드바 6 항목, 디자인 핸드오프 변형 5개를 6 라우트에 분배.

| 메뉴 | 라우트 | 디자인 출처 |
|------|--------|-------------|
| 프로필 | `/settings/account` | A 상단 카드 (이름·이메일·휴대폰·구성원) |
| 한전 연동 | `/settings/account` | A 하단 카드 (고객번호·주소·계약·연동일) |
| 알림 | `/settings/notifications` | B (알림×채널 매트릭스 + 방해 금지) |
| 보안 | `/settings/security` | C (비밀번호/2FA/세션/위험 영역) |
| 이상 탐지 내역 | `/settings/anomaly-log` | D (KPI 3 + 필터 pill + 이벤트 테이블 + CSV/JSON/PDF) |
| 이메일 연동 | `/settings/email` | E (SMTP/POP + 수신자 + 발송 규칙 + 템플릿 + 테스트) |

프로필과 한전 연동은 같은 라우트(`/settings/account`)에 카드 2개로 표시.

## 요구사항 매핑 (REQ-001~009)

| REQ | 영역 | 화면 |
|-----|------|------|
| REQ-001 | NILM 분해 | 04 사용량 분석 |
| REQ-002 | 이상 탐지 | **07 AI 진단 + 06-D 이상 탐지 내역** (의도적 분산) |
| REQ-003 | 캐시백 (KEPCO 피벗) | 05 캐시백 |
| REQ-006 | UI | 7화면 전체 |
| REQ-007 | 인증 | 02 로그인/회원가입 |

### REQ-002 분산 의도 (확정)

- **07 AI 진단 (`/insights`)** — 실시간 인사이트·추천. 이상 사용 즉시 카드, 절약 추천, NILM 모델 신뢰도, 주간 진단 추이.
- **06-D 이상 탐지 내역 (`/settings/anomaly-log`)** — 과거 로그·감사용. 전체 이벤트 테이블, 기간/심각도/가전/상태 필터, CSV/JSON/PDF 내보내기, 이메일 발송 이력.
- 06-E 이메일 연동은 06-D 의 알림 채널 설정.

## 기술 스택 (디자인 토큰 매핑 시 참조)

스택: Vite 5 + React 18 + TS 5 + Tailwind CSS 3 + Recharts + React Router v6 + TanStack Query v5 + Zustand + Axios.

디자인 핸드오프의 디자인 토큰(README §Design Tokens)을 Tailwind config 에 그대로 매핑. 그레이스케일 베이스(`--bg`, `--canvas`, `--ink-1~4`, `--line-1~3`, `--fill-1~3`) + Inter / JetBrains Mono.
