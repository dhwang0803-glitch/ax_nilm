# 에너지캐시백 기준 사용량 산정 방식

## 기준 사용량이란?

에너지캐시백에서 절감 여부를 판단하는 기준이 되는 전력 사용량이다.
당월 실제 사용량과 비교해 기준 사용량보다 얼마나 줄였는지를 계산한다.

---

## 기본 산정 방식 — 동월 2개년 평균

```
기준 사용량(kWh) = (작년 동월 사용량 + 재작년 동월 사용량) ÷ 2
```

### 예시

| 항목 | 사용량 |
|------|--------|
| 2022년 7월 사용량 | 380 kWh |
| 2023년 7월 사용량 | 420 kWh |
| **2024년 7월 기준 사용량** | **(380 + 420) ÷ 2 = 400 kWh** |

- "동월"은 검침일 기반 청구 사이클이 같은 달을 의미한다.
- 두 개년 모두 이력이 있어야 이 방식이 적용된다.

---

## 신규 가구 산정 방식 — 군집(클러스터) 평균 대체

과거 2개년 사용 이력이 없는 신규 세대는 유사 가구 집합의 평균으로 기준을 설정한다.

### 군집 분류 기준

유사 가구는 다음 속성을 기반으로 묶인다.

| 속성 | 예시 |
|------|------|
| 주거 유형 | 아파트, 단독주택, 연립/다세대 |
| 월평균 사용량 구간 | 저소비(~150 kWh), 중소비(150~350 kWh), 고소비(350 kWh~) |
| 지역 기후대 | 중부권, 남부권 |

```
기준 사용량 = 동일 군집 내 같은 월 평균 사용량
```

### 군집 대체 적용 기간

- 이력 0개년: 전적으로 군집 평균 적용
- 이력 1개년: 해당 1개년 실측값과 군집 평균의 혼합 (운영 정책에 따라 비중 조정)
- 이력 2개년 이상: 동월 2개년 평균으로 전환

---

## 기준 사용량에 영향을 주는 요소

### 과거 사용량이 높으면 유리하다

기준이 높을수록 실제 사용량과의 차이가 벌어지기 쉬워 절감률을 달성하기 쉽다.
반대로 과거에 이미 절전을 잘 하던 가구는 기준이 낮아 추가 절감 여지가 적다.

### 계절성이 반영된다

동월 비교이므로 여름에는 여름 기준, 겨울에는 겨울 기준이 적용된다.
폭염·혹한으로 과거 사용량이 높았던 해가 기준연도에 포함되면 기준 사용량이 높아진다.

---

## 절감률 계산

```
절감량(kWh) = 기준 사용량 - 당월 실제 사용량
절감률(%)   = 절감량 ÷ 기준 사용량 × 100
```

- 절감률이 **3% 미만**이면 캐시백이 발생하지 않는다.
- 절감률이 **30% 초과**이면 30% 절감분까지만 캐시백을 산정한다.

### 절감률 구간별 단가

| 절감률 | 단가 |
|--------|------|
| ≥ 20% | 100원/kWh |
| ≥ 10% | 80원/kWh |
| ≥ 5% | 60원/kWh |
| ≥ 3% | 30원/kWh |

## 자주 묻는 질문

**Q. 작년에 이사 와서 이력이 1개년뿐입니다. 어떻게 되나요?**
이력 1개년과 군집 평균을 조합해 기준 사용량이 설정된다. 다음 해부터는 자체 2개년 이력으로 전환된다.

**Q. 리모델링으로 가전을 교체해 사용량이 크게 줄었습니다. 기준이 불리하지 않나요?**
기준은 과거 2개년 평균이므로 교체 전 높은 사용량이 기준에 반영된다. 오히려 절감률이 쉽게 달성될 수 있어 유리하다.

**Q. 여름에 기준 사용량이 겨울보다 높은 이유는?**
동월 비교 방식이므로 여름 기준은 전년도 여름 평균이 적용된다. 냉방 수요가 많은 여름에는 기준 자체도 높게 설정된다.

**Q. 전기차 충전으로 사용량이 늘었는데 불이익이 있나요?**
주택용 저압 계량기 기준이므로 전기차 충전량이 포함되면 기준 사용량과 실제 사용량 모두 증가한다. 충전량을 제외한 순수 생활 전력은 줄지 않더라도 전체 사용량 증가로 캐시백 달성이 어려워질 수 있다.
