# 에너지캐시백 신청 방법 및 자격 조건

## 에너지캐시백이란?

KEPCO(한국전력공사)가 운영하는 전기 절감 인센티브 프로그램이다.
기준 사용량보다 3% 이상 전기를 절감하면 절감량에 비례해 전기요금을 자동 차감해준다.
별도 기기 설치나 복잡한 절차 없이 신청만으로 참여할 수 있다.

---

## 신청 자격

| 조건 | 내용 |
|------|------|
| 계약 종류 | 주택용 저압 전력 계약 |
| 사용 형태 | 개별 계량기 보유 세대 (아파트 세대 포함) |
| 이력 조건 | 과거 사용 이력 없어도 신규 신청 가능 (군집 평균 기준 적용) |
| 중복 참여 | 동일 고객번호로 중복 신청 불가 |

- **고압 계약** 아파트(동 전체를 하나로 계량하는 단지)는 별도 집합 신청 절차 필요.
- 기초생활수급자 등 복지 할인 대상자도 신청 가능하며 캐시백은 복지 할인과 별도 적용된다.

---

## 신청 방법

### 방법 1 — 온라인 신청 (권장)

1. KEPCO 에너지캐시백 신청 페이지 접속: `https://en-ter.co.kr/ec/apply/prsApply/select.do`
2. 공동인증서 또는 간편인증(카카오·네이버·PASS 등)으로 본인 인증
3. 고객번호 확인 (전기요금 고지서 또는 한전ON 앱에서 확인 가능)
4. 신청 정보 입력 후 제출
5. 완료 후 SMS/이메일 확인

### 방법 2 — 한전ON 앱 신청

1. 한전ON 앱 설치 (Android / iOS)
2. 회원가입 후 고객번호 등록
3. 앱 내 "에너지캐시백" 메뉴 → 신청

### 방법 3 — 한전 고객센터

- 전화: 국번 없이 123
- 방문: 가까운 한전 지사

---

## 신청 시 필요 정보

| 항목 | 확인 방법 |
|------|----------|
| 고객번호 (10자리) | 전기요금 고지서 상단, 한전ON 앱 |
| 주민등록번호 앞 6자리 | 본인 인증용 |
| 연락처 | 결과 통보용 |

---

## 신청 후 프로세스

```
신청 완료
    ↓
첫 번째 검침 사이클 완료 대기 (신청 월 기준 1~2개월)
    ↓
기준 사용량 확정 (동월 2개년 평균 또는 군집 평균)
    ↓
매월 절감량 자동 산정
    ↓
익월 전기요금 청구서에서 캐시백 자동 차감
```

---

## 캐시백 확인 방법

- **한전ON 앱**: 에너지캐시백 메뉴 → 월별 절감량·캐시백 내역
- **전기요금 고지서**: "에너지캐시백 차감" 항목
- **KEPCO 사이버지점**: cyber.kepco.co.kr 로그인 → 요금조회

## 자주 묻는 질문

**Q. 에너지캐시백은 따로 돈을 받는 건가요?**
별도 입금이 아니라 다음 달 전기요금에서 자동으로 차감된다. 고지서에 "에너지캐시백" 항목으로 표시된다.

**Q. 신청하면 무조건 캐시백을 받나요?**
기준 사용량보다 3% 이상 절감해야 캐시백이 발생한다. 절감 없이는 지급되지 않지만 불이익도 없다.

**Q. 이사하면 신청이 유지되나요?**
이사로 고객번호가 바뀌면 기존 신청이 종료되고 새 주소에서 재신청해야 한다.

**Q. 신청을 취소할 수 있나요?**
언제든지 한전ON 앱 또는 고객센터(123)를 통해 참여 해지 가능하다.
